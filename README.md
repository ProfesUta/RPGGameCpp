# RPGGameCpp

A simple **console-based RPG game written in C++**, built step by step as a learning project.

This project is focused on learning:
- Core C++ syntax
- Project structure (.h / .cpp separation)
- Game logic (combat, XP, leveling)
- Clean incremental development

---

## Features

- Create a player with custom name
- Turn-based combat system
- Enemies with basic AI
- Randomized damage
- XP & level-up system
- Stat progression (HP, attack, level)

---

## Built With

- **C++**
- **Visual Studio Community 2026**
- Standard C++ libraries only (no external dependencies)
